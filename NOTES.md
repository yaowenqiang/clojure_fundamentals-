# Operation Forms

+ ( op ..)
  + op can be one of:
    + Special operators macro
    + Expression that yields a function
    + Something invocable

# Literals

+ 42          ; Long
+ 6.022e23    ; Double
+ 42N         ; BigInt
+ 1.0M        ; BigDecimal
+ 22/7        ; Ratio
+ "hello"     ; String
+ \e          ; Character
+ true false  ; Booleans
+ nil         ; null
+ Fred *bob*  ; Symbols
+ :alpha :beta ; Keywords


## Data Structures

+ (4 : alpha 3.0)      ; List
+ [2 "Hello" 99]       ; Vector
+ {:a 1, :b 2}         ; Map
+ {:a 1 :b 2}          ; Map
+ #{alice jim bob}     ; Set
